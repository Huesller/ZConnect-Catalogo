# ZConnect Catálogo

Catálogo B2B automotivo premium da Z Connect.

## Requisitos

- Node.js
- NPM
- Git

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Atualização do catálogo

```bash
npm run update-daily
```

## DevKit

Os comandos de rotina estão em:

```txt
dev/
```

Atalho principal:

```txt
ZCONNECT.bat
```

## Publicação

```txt
dev/PUBLICAR.bat
```

O deploy é feito automaticamente pela Vercel após o push no GitHub.
