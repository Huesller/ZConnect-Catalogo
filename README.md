# Z Connect Catálogo Premium

Catálogo B2B automotivo em React + Vite.

## Comandos

```bash
npm install
npm run dev
npm run build
npm run update-daily
```

## Deploy

Projeto pronto para GitHub + Vercel.

Arquivos versionados:
- `src/`
- `public/`
- `scripts/`
- `package.json`
- `package-lock.json`
- `index.html`

Arquivos ignorados:
- `node_modules/`
- `dist/`
- `.vercel/`
- `.git/`
- zips e backups locais
