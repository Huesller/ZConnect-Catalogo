# Z Connect DevKit V3

Copie `ZCONNECT.bat` e a pasta `dev/` para a raiz do projeto `ZConnect-Catalogo`.

Abra `ZCONNECT.bat` com duplo clique.

Primeira vez na máquina: use a opção `1 - Primeira instalacao / Setup`.

Depois use:
- `2 - Build inteligente`
- `3 - Publicar no GitHub`
- `5 - Status completo`
