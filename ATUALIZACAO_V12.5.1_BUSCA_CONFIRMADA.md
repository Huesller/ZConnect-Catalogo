# Atualização 12.5.1 — busca confirmada

- Digitar no campo continua exibindo sugestões, mas não grava eventos.
- A busca é registrada somente quando o cliente pressiona Enter.
- Clicar em uma sugestão registra apenas a abertura daquele produto específico.
- Fragmentos digitados, como `lan`, não geram demanda nem busca sem resultado.
