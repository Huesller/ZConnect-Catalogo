# Publicação do Catálogo

1. Configure as variáveis indicadas no `.env.example` e na Vercel.
2. Execute `npm test`.
3. Execute `npm run build`.
4. Confirme que o sincronizador informa envio bem-sucedido do snapshot quando as credenciais estão configuradas.
5. Publique na Vercel e valide busca, carrinho, cotação e identificação da empresa.

Esta limpeza não altera regras de preço, reserva, busca confirmada ou rastreamento comercial.
